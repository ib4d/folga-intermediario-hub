# Segment Message Angles

Use these angles so the same message is not sent to everyone.

## 1. Founder / Owner

Primary angle:

- operational control
- scalable process
- less chaos as volume grows

Use:

`When agencies scale internationally, Excel and WhatsApp stop giving real control over documents, legal follow-up, and arrivals.`

## 2. Operations Manager

Primary angle:

- traceability
- coordination
- fewer broken handoffs

Use:

`The real bottleneck is usually not candidate volume alone, but coordination across recruiters, documents, legal, and logistics.`

## 3. Recruitment Lead

Primary angle:

- less manual chasing
- cleaner flow
- fewer document errors

Use:

`Recruiters lose too much time chasing documents and retyping fields that should already be in the system.`

## 4. Legal / Documentation Coordinator

Primary angle:

- document reliability
- review discipline
- expiry and status visibility

Use:

`Document handling gets risky when passport, permit, and PESEL review live across chats and spreadsheets.`

## 5. Logistics Coordinator

Primary angle:

- arrival readiness
- handoff clarity
- less last-minute confusion

Use:

`Logistics gets noisy when arrivals, document readiness, and internal follow-up are not tied together in one operational flow.`

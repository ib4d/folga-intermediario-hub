# Commercial Execution Playbook

This is the practical playbook for the first commercial push.

It is built around guided selling, not broad marketing automation.

## 1. Target account strategy

Use a tight account list first.

Recommended first batch:

- 20 target agencies
- 1-3 contacts per agency
- 20-40 total decision-maker contacts

## 2. Account selection rules

Prioritize agencies that show:

- international worker recruitment
- visible growth
- multiple recruiters/operators
- document-heavy operations
- likely Excel/WhatsApp dependency

Deprioritize agencies that look like:

- one-person brokers with minimal process
- pure job-board style businesses
- companies with no visible operational complexity

## 3. Contact segmentation

Segment A:

- founder / owner
- message angle: control, scale, visibility

Segment B:

- operations manager
- message angle: workflow, traceability, coordination

Segment C:

- recruitment lead
- message angle: less manual chasing, cleaner candidate flow

## 4. First 4-week cadence

### Week 1

- build target list
- write contact notes
- send first outbound wave

### Week 2

- send follow-up wave
- book discovery/demo calls

### Week 3

- run demos
- qualify pilot readiness

### Week 4

- present pilot offer
- close pilot or disqualify

## 5. Daily output target

Reasonable disciplined target:

- 5-10 good outbound touches per day
- not 50 low-quality generic touches

The first wave should optimize for quality and learning, not spam volume.

## 6. Suggested CRM tracking fields

Track at minimum:

- account name
- contact name
- role
- country
- segment
- current pain
- status
- next action date
- owner

## 7. Pipeline stages

Use this sequence:

1. target identified
2. first contact sent
3. replied
4. discovery booked
5. demo completed
6. pilot qualified
7. proposal sent
8. pilot won
9. rollout won
10. disqualified / lost

## 8. Disqualification rules

Disqualify if:

- no owner exists on customer side
- no visible operational pain
- only wants free exploration
- insists on fully self-serve motion now
- price conversation has no seriousness

## 9. Weekly review questions

Every week, ask:

- which messages get replies
- which roles convert better
- where prospects hesitate
- where pricing resistance appears
- which accounts deserve deeper follow-up

## 10. Recommended current motion

Use:

`Small account list, high-fit prospects, guided demo, disciplined qualification, and strong pricing control.`

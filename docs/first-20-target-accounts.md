# First 20 Target Accounts

This is the first practical target-account list for ORI CRUIT HUB.

It is intentionally selective.

The goal is not to create a giant database. The goal is to create the first
high-fit account set for guided outreach.

## Selection logic

Accounts were prioritized when they show some combination of:

- staffing / temporary work scale
- employer-facing recruitment services
- foreign worker handling
- legalization / work permit support
- operational complexity across recruiting, legal, and logistics

# Roles y Permisos

Folga Hub utiliza un sistema de control de acceso basado en roles (RBAC).

## Definición de Roles

### 1. SUPERADMIN
- Acceso total al sistema.
- Gestión de usuarios y roles.
- Eliminación y anonimización de datos.
- Exportación total de auditoría.

### 2. ADMIN
- Gestión de candidatos de todos los intermediarios.
- Configuración del sistema.
- Visualización de reportes generales.

### 3. LEGAL
- Foco en la validación de documentos.
- Acceso al panel de "Revisión Legal".
- Poder de Aprobar/Rechazar candidatos para trámites migratorios.
- Visualización de datos de identidad completos (sin máscara).

### 4. LOGISTICA
- Gestión de llegadas y transporte.
- Acceso al panel de "Logística".
- Confirmación de recogidas y alojamiento.

### 5. INTERMEDIARIO
- Creación y seguimiento de sus propios candidatos.
- Subida de documentos.
- Visualización limitada de datos (máscara en PESEL/Pasaporte de candidatos ajenos).
- Recepción de notificaciones sobre el estado de sus candidatos.

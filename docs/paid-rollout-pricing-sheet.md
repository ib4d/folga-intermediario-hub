# Paid Rollout Pricing Sheet

This document defines the pricing logic after a pilot succeeds.

It is intentionally simple and should be used as a commercial working sheet,
not as polished public pricing copy yet.

## 1. Purpose

Paid rollout pricing should:

- convert validated pilots into real customers
- reflect the operating value of the platform
- stay compatible with a guided rollout model

## 2. Recommended structure

Use a structure with:

1. recurring platform fee
2. optional onboarding/expansion fee when scope increases

Optional later:

- tiering by operator count
- tiering by candidate/document volume
- add-on services

## 3. What the paid rollout should cover

At minimum:

- active production tenant
- defined operator access
- production support expectations
- agreed workflow scope
- routine production use

## 4. What should remain separate

Keep these outside base pricing unless intentionally included:

- major customizations
- unusual migration work
- non-standard support expectations
- extensive training cycles

## 5. Recommended internal pricing fields

Recommended tenant-based pricing:

### Core

- monthly base fee: `€1,250 / month`
- included seats/operators: `up to 5`
- included scope: `1 tenant, core candidate/doc/legal/logistics workflow`

### Growth

- monthly base fee: `€2,500 / month`
- included seats/operators: `up to 15`
- included scope: `broader team rollout, stronger operating cadence`

### Scale

- monthly base fee: `€4,500 / month`
- included seats/operators: `up to 30`
- included scope: `multi-team operational rollout with higher-touch support`

### Enterprise

- pricing: `from €7,500 / month`
- scope: `custom commercial review`

Expansion logic:

- additional operator pack: `€300 / month per 5 operators`
- additional tenant / brand environment: `from €750 / month`
- premium onboarding / migration project: `from €2,000 one-time`

Optional add-ons:

- branded rollout / white-label setup: `from €1,500 one-time`
- custom reporting / export package: `from €500 / month`
- higher-touch support cadence: `from €750 / month`

## 6. Positioning line

Use:

`After the pilot proves operational value, we move into a paid rollout with a defined production scope and commercial model.`

## 7. Current decision

Recommended motion:

`Convert successful pilots into a paid rollout with explicit scope, instead of forcing a premature self-serve pricing model.`

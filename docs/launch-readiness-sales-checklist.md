# Launch Readiness Sales Checklist

Use this checklist before treating the current commercial package as actively
ready for repeated outbound execution.

The goal is operational readiness, not theoretical completeness.

## 1. Core commercial assets

These should already be available and usable:

- [ ] one approved narrative
- [ ] one approved one-pager
- [ ] one approved commercial deck
- [ ] one approved pricing structure
- [ ] one approved pilot framing
- [ ] one short proposal template

## 2. Outreach assets

- [ ] first target list exists
- [ ] first CRM working sheet exists
- [ ] first outreach batch exists
- [ ] follow-up sequence exists
- [ ] weekly tracker exists

## 3. Demo and qualification assets

- [ ] demo story is stable
- [ ] discovery/demo notes template exists
- [ ] demo-to-pilot qualification checklist exists
- [ ] pipeline stage rules exist

## 4. Proposal and closing assets

- [ ] proposal control rules exist
- [ ] pricing objection handling exists
- [ ] discount discipline exists
- [ ] pilot handoff template exists

## 5. Operational sale constraints are clear

- [ ] product is positioned as guided pilot, not self-serve SaaS
- [ ] OCR/manual-review limitations are framed honestly
- [ ] pilot scope expectations are narrow and explicit
- [ ] rollout pricing is separated from pilot pricing

## 6. Demo / pilot environment readiness

- [ ] demo tenant path is known
- [ ] demo reset path is known
- [ ] pilot onboarding handoff path is known
- [ ] production status is stable enough for guided pilots

## 7. First execution discipline

- [ ] one owner of outreach execution is clear
- [ ] one owner of demos is clear
- [ ] one owner of proposal follow-up is clear
- [ ] weekly review cadence is defined

## 8. Go / no-go rule

You are commercially ready to execute if:

- the assets exist
- the pricing is controlled
- the stage language is consistent
- the next 10-20 account wave can be run without improvisation

If those conditions are true, do not wait for perfect polish to start.

## 9. Rule to remember

Commercial readiness means:

- you can run the motion repeatedly
- you can qualify honestly
- you can close without inventing process mid-flight

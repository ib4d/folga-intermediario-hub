# Short Commercial Proposal

## ORI CRUIT HUB

### Proposal type

Guided pilot for international recruitment operations.

## What is included

- 1 guided pilot tenant
- up to 5 operators
- guided onboarding
- 1 defined workflow scope
- 30-day validation cycle
- weekly review cadence

## Commercial terms

Standard pilot:

- setup fee: `EUR 2,500`
- pilot cycle fee: `EUR 2,000 / 30 days`

If the pilot succeeds, paid rollout starts from:

- `EUR 1,250 / month`

## What the pilot is for

The pilot is designed to validate:

- candidate workflow adoption
- document upload and OCR-assisted review
- legal/logistics follow-up in the app
- operational fit with the customer team

## What is not included by default

- unlimited support
- custom development
- unusual migration scope
- white-label rollout

These can be priced separately if needed.

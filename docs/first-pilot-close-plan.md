# First Pilot Close Plan

This document defines how to handle the first serious opportunity from
pilot-qualified account to signed guided pilot.

The objective is not just to send a proposal.

The objective is to create a controlled first commercial win.

## 1. What the first pilot should look like

The first pilot should be:

- narrow
- operationally real
- commercially paid
- easy to support
- credible as a future reference

Do not try to win the first pilot with oversized scope.

## 2. Ideal first-pilot profile

Prefer an account with:

- obvious document-heavy operations
- visible cross-team coordination pain
- one reachable operational owner
- one realistic decision path
- manageable complexity for a 30-day pilot

Best first wins are not the biggest logos.

They are the cleanest closable logos.

## 3. What must be true before trying to close

- pain has been validated
- workflow in scope is concrete
- internal owner on buyer side is visible
- pilot outcome is understandable
- proposal has been intentionally framed
- commercial next step is explicit

If one of these is weak, closing pressure should not increase yet.

## 4. Closing sequence

Recommended sequence:

1. run qualified demo
2. confirm fit and pilot scope
3. send proposal
4. run proposal review / decision call
5. resolve scope or pricing issues
6. get explicit yes
7. hand off into guided onboarding

Avoid trying to close only by email.

## 5. What to protect during the first close

Protect:

- price integrity
- scope discipline
- implementation realism
- onboarding control

Do not win the first deal by creating an operationally bad pilot.

## 6. Acceptable concession logic

If concession is needed:

1. reduce scope first
2. reduce optional support first
3. use early-adopter pricing only if strategically justified

Do not trade away margin before tightening scope.

## 7. What the proposal review call must achieve

The proposal review call should clarify:

- what is included
- what success looks like
- who owns the pilot
- what the target start date is
- what blocker remains, if any

If none of these move forward, the call did not do enough.

## 8. Strong closing signals

Treat these as positive indicators:

- buyer invites operational stakeholder
- they ask about start timing
- they ask about onboarding steps
- they ask about pilot success criteria
- they discuss internal approval mechanics concretely

## 9. Weak closing signals

Treat these as warning signs:

- they avoid live review of the proposal
- they ask for lower pricing without confirming scope
- they delay decisions without naming blocker
- they keep interest high but commitment low

## 10. Definition of first-win success

A successful first pilot win means:

- the deal is paid
- the scope is controlled
- the client owner is explicit
- the start path is clear
- the handoff is clean

That matters more than maximizing first-deal size.

## 11. Rule to remember

The first pilot should validate the commercial motion, not distort it.

# Target Account List Template

Use this template to build the first commercial target list.

## Required fields

| Field | Notes |
|------|------|
| Account name | Agency name |
| Country | Operating market |
| Website | Public URL |
| Segment | Founder / Ops / Recruitment lead |
| Contact name | Primary contact |
| Role | Decision-maker role |
| LinkedIn URL | Contact profile |
| Email | If available |
| ICP fit | High / Medium / Low |
| Pain hypothesis | Why ORI CRUIT HUB may fit |
| Status | Current pipeline status |
| Next step | Immediate next action |
| Notes | Context |

## Suggested first statuses

- target identified
- first contact sent
- follow-up sent
- replied
- discovery booked
- demo completed
- pilot qualified
- proposal sent
- won
- lost

## Rule

Do not fill this with random low-fit names.

The first version should be short, specific, and high-fit.

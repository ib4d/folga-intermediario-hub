# Day 1 Outreach Checklist

Use this checklist on the first real day of outbound execution.

The goal is simple:

- do not improvise
- do not overbuild
- do not lose discipline in the first wave

## 1. Before sending anything

Confirm:

- [ ] the Tier A CRM sheet is open
- [ ] the first outreach batch is ready
- [ ] the owner of outreach execution is clear
- [ ] each account has one current stage
- [ ] each account has one next action

## 2. Accounts to touch first

Start with:

1. Personnel Service
2. ATERIMA WORK
3. EWL Group
4. Gremi Personal

Do not start with all 8 at once if you cannot track them properly.

## 3. Minimum assets to keep open while sending

Keep open:

- `docs/first-crm-sheet-tier-a.csv`
- `docs/first-outreach-batch-tier-a.md`
- `docs/follow-up-sequence-tier-a.md`
- `docs/weekly-outreach-tracker-template.md`

## 4. Send discipline

For each outbound touch:

- [ ] use the right account-specific message
- [ ] personalize the greeting
- [ ] keep the pain hypothesis intact
- [ ] do not mention pricing unless fit is already acknowledged
- [ ] log the send immediately after sending

## 5. CRM update rule

Immediately after sending:

- change status to `first contact sent`
- add last action date
- add next action date
- set next action to `Follow up in 3-5 working days`

Do not leave any touched account without the next step recorded.

## 6. If a reply comes in

Do this immediately:

- [ ] mark status as `replied`
- [ ] summarize the reply in notes
- [ ] decide whether the reply is:
  - strong
  - vague
  - weak
- [ ] set next step:
  - book discovery
  - clarify pain
  - disqualify

## 7. If pricing comes up too early

Use the approved rule:

- [ ] do not negotiate immediately
- [ ] do not send full pricing policy
- [ ] reframe toward workflow fit first

## 8. End-of-day review

At the end of day 1, record:

- total first touches sent
- total replies
- which message felt strongest
- which account felt most promising
- which account needs better routing

## 9. What success looks like on day 1

Success on day 1 is not a closed deal.

Success on day 1 means:

- the first wave is sent cleanly
- the CRM is updated
- no account is left without next action
- the process is ready for follow-up discipline

## 10. Day 1 operating rule

Use:

`A clean first day beats a bigger but sloppy first day.`

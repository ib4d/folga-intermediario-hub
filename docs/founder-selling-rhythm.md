# Founder Selling Rhythm

This document defines the minimum weekly rhythm for founder-led commercial
execution at the current stage of ORI CRUIT HUB.

The goal is consistency.

Not intensity without control.

## 1. Weekly structure

Use a simple weekly rhythm:

- Monday: pipeline review and account prioritization
- Tuesday: first outreach and follow-up execution
- Wednesday: demos / discovery / proposal work
- Thursday: follow-up, proposal control, close movement
- Friday: review, cleanup, and next-week planning

## 2. Monday

Every Monday:

- review all active accounts
- remove stale optimism
- select this week’s priority accounts
- confirm next actions
- confirm which opportunities deserve live effort

Output:

- one clean active list
- one outreach list
- one follow-up list
- one close-focus list

## 3. Tuesday

Use Tuesday for:

- first outbound sends
- first follow-ups
- CRM updates
- contact route cleanup

Rule:

- sending without logging is not allowed

## 4. Wednesday

Use Wednesday for:

- discovery calls
- demos
- post-demo notes
- pilot qualification decisions

Rule:

- every live conversation must end in one clear next step

## 5. Thursday

Use Thursday for:

- proposal sending
- proposal review follow-ups
- pricing control
- decision-path forcing

This should be the most commercially disciplined day.

## 6. Friday

Use Friday for:

- reviewing what actually moved
- checking stale accounts
- updating tracker metrics
- deciding nurture / disqualify actions
- preparing next week’s top accounts

Do not carry unclear pipeline into the next week.

## 7. Daily minimum questions

At the end of each working day ask:

1. Which account moved one real step?
2. Which account was touched but not updated properly?
3. Which opportunity is weaker than it looked this morning?
4. What is tomorrow’s most important commercial action?

## 8. Minimum weekly metrics

Track only:

- new first contacts sent
- follow-ups sent
- replies received
- demos booked
- demos completed
- pilot-qualified opportunities
- proposals sent
- wins

## 9. Founder focus rule

Founder time should go mostly to:

- strongest-fit accounts
- best live conversations
- pricing discipline
- first closes

Do not spend founder time equally across weak and strong accounts.

## 10. Anti-chaos rules

Avoid these habits:

- random outreach bursts
- demos without prep
- pricing before qualification
- unlogged follow-ups
- stale decision pending accounts

## 11. Rule to remember

Commercial rhythm should create:

- visible weekly progress
- honest pipeline
- repeatable closing behavior

That is enough for this stage.
